CASIO 9850G Text to FA-122 code converter.

To use this converter, use any program from this the CASIO FA-122 Software Listing, then copy/paste it into the text area shown below then press "Text to FA-122" button and voilá.

Press the "Text to FA-122" button and copy it to the FA-122 or FA-123 programs or press "Write to symbols" in order to see the program with symbols. This is a preeliminary version but it seems to work well, any bugs write to xavier [at] supositorio.com

This script is intended to take the doublespaces and comments away. It also changes the characters shown in the table above into their repective symbols. For any questions or suggestions please write to xavier a+ supositorio.com
